AI Gesture-Controlled 3D Particle Visualization
Technologies: HTML, CSS, JavaScript, Three.js, MediaPipe Hands, WebGL

1. Developed an interactive 3D particle visualization system rendered with 8,000 particles forming shapes like heart, flower, Saturn, and fireworks.

2. Integrated AI hand-tracking using MediaPipe to control particle expansion and contraction through real-time hand gestures.

3. Implemented smooth animations and user controls such as shape selection and color customization using JavaScript and Three.js.
